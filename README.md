### 🔗 [Visual Chain](https://github.com/Hamza1610/bitcoin_block_weave) — Creator & Lead Developer
> *Interactive Bitcoin fork, reorg & chain creator for Regtest and Signet*

The first truly interactive tool for creating and observing Bitcoin blockchain forks and chain reorganizations in real time. While tools like fork.observer offer passive observation, Visual Chain lets developers mine blocks on any chain tip, build competing chains, and trigger reorgs with one click — watching the node switch live in a directed graph visualization.

Built for Lightning devs, wallet teams, and educators who need to test reorg handling without CLI-only workflows. Currently local — deployment planned.

- Real-time DAG/tree visualization of all chain tips and forks
- Color-coded chain status: Active (green), Valid-Fork (orange), Stale (gray), Invalid (red)
- One-click "Mine here" on any block tip
- Create Fork tool — mine competing chains of chosen depth
- Live reorg detection with smooth animation and highlight
- Connects to local Regtest and Signet nodes

`TypeScript` `Bitcoin Core RPC` `Regtest` `Signet`
